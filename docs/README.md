# Crucible Turbine Role Selection Form (CTRSF)

This is the documentation website for the Crucible Turbine project, hosted on GitHub Pages.

## About

This website provides information about the Crucible Turbine Role Selection Form (CTRSF), a high-integrity project management system built on the principle of Strategic Integrity.

The website is accessible at: https://crucible-turbine-role-form.github.io
